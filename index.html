A* Pathfinding Visualization

This Python project implements the A* path-finding algorithm to find the shortest path between a start and end point on a grid. The program utilizes Pygame to visualize the algorithm in real-time. The program creates a grid of Spots that are either empty, closed, or barriers. The algorithm uses a priority queue to determine the shortest path to the end point, while also avoiding barriers. The visualization displays the search progress and the final shortest path.
